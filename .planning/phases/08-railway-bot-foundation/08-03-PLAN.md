    1. Create `src/pipeline/bot/dispatcher.py`:
       - `async def trigger_pipeline(token: str, owner: str, repo: str) -> bool`:

         - Headers: `Accept: application/vnd.github.v3+json`, `Authorization: Bearer {token}`
         - JSON payload: `{"event_type": "run_now", "client_payload": {"triggered_by": "telegram_bot"}}`
         - Use `httpx.AsyncClient(timeout=15.0)`
         - Return `resp.status_code == 204`
         - Wrap in try/except to catch httpx errors, log warning, return False. Never crash.

       - `async def run_now_command(update: Update, context: ContextTypes.DEFAULT_TYPE) -> None`:
         - Read GITHUB_PAT, GITHUB_OWNER, GITHUB_REPO from env vars.
         - If any are missing, reply "GitHub integration not configured. Set GITHUB_PAT, GITHUB_OWNER, GITHUB_REPO env vars." and return.
         - Reply "Triggering pipeline run..." first (immediate feedback).
         - Call `trigger_pipeline(token, owner, repo)`.
         - If True: reply "Pipeline run dispatched. Check GitHub Actions for progress."
         - If False: reply "Failed to dispatch pipeline run. Check bot logs."

       - Update the help_command import if needed (it's already imported).
