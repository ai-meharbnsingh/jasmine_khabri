**2. Create shared hashing utilities** (`src/pipeline/utils/hashing.py`):

```python
"""Shared text normalization and hashing for filter pipeline."""
import hashlib
import re
import unicodedata

_PUNCT_RE = re.compile(r"[^a-z0-9\s]")
_SPACE_RE = re.compile(r"\s+")


def normalize_title(text: str) -> str:
    """Lowercase, NFD-normalize, strip punctuation, collapse whitespace."""
    nfd = unicodedata.normalize("NFD", text.lower())
    cleaned = _PUNCT_RE.sub(" ", nfd)
    return _SPACE_RE.sub(" ", cleaned).strip()


def compute_title_hash(title: str) -> str:
    """Return SHA-256 hex digest of normalized title."""
    return hashlib.sha256(normalize_title(title).encode("utf-8")).hexdigest()
```

This is the canonical normalization function. Phase 4 dedup and all future phases MUST import from here.

**4. Create relevance_filter.py** (`src/pipeline/filters/relevance_filter.py`):

Two pure functions:

`score_article(article: Article, keywords: KeywordsConfig) -> tuple[bool, int]`:
- Concatenate `f"{article.title} {article.summary}".lower()` as search text
- **Exclusion check first** (short-circuit): iterate `keywords.exclusions`, if any exclusion keyword (lowercased) is found in text via `in` operator, return `(False, 0)`
- **Scoring**: iterate `keywords.active_keywords()`, for each keyword (lowercased):
  - If keyword found in `article.title.lower()`: add 20 points
  - Elif keyword found in text (description-only match): add 10 points
  - Each keyword counted at most once (title match takes priority over body)
- Return `(True, score)`

`filter_by_relevance(articles: list[Article], keywords: KeywordsConfig, threshold: int = 40) -> list[Article]`:
- For each article, call `score_article()`
- If `passes_exclusion` is False, skip (filtered out)
- If `relevance_score < threshold`, skip
- Otherwise, return a copy of the article with `relevance_score` set: `article.model_copy(update={"relevance_score": score})`
- Return the list of passing articles
- Log count: `"Relevance filter: %d/%d articles passed (threshold=%d)"` at INFO level

Import Article from `pipeline.schemas.article_schema`, KeywordsConfig from `pipeline.schemas.keywords_schema`. Use logging module for the log line.
