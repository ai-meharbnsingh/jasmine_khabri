**City taxonomy** (frozensets for O(1) lookup):

```python
TIER_1_CITIES = frozenset({
    "delhi", "delhi ncr", "ncr", "mumbai", "bangalore", "bengaluru",
    "hyderabad", "chennai", "kolkata", "pune", "ahmedabad",
})

TIER_2_CITIES = frozenset({
    "noida", "gurugram", "gurgaon", "faridabad", "ghaziabad",
    "jaipur", "surat", "lucknow", "kanpur", "nagpur", "indore",
    "bhopal", "visakhapatnam", "patna", "vadodara", "thane", "navi mumbai",
})

GOV_SOURCES = frozenset({
    "mohua", "nhai", "aai", "smart cities",
})
```

**`classify_geo_tier(article: Article) -> int`:**
- Concatenate `f"{article.title} {article.summary}".lower()` as search text
- Scan for any Tier 1 city in text (use `in` operator): if found, return 1
- Scan for any Tier 2 city in text: if found, return 2
- If NO city found: check if `article.source.lower()` is in GOV_SOURCES -> return 1 (national-scope government article per research Pitfall 4 recommendation)
- Otherwise return 3

**`filter_by_geo_tier(articles: list[Article], tier2_threshold: int = 60, tier3_threshold: int = 85) -> list[Article]`:**
- For each article:
  - Classify tier via `classify_geo_tier(article)`
  - Tier 1: always include
  - Tier 2: include only if `article.relevance_score >= tier2_threshold` (HIGH proxy -- documented as Phase 5 refinement point)
  - Tier 3: include only if `article.relevance_score >= tier3_threshold`
  - Set `geo_tier` on passing articles: `article.model_copy(update={"geo_tier": tier})`
- Log: `"Geo filter: %d/%d articles passed (T1:%d T2:%d T3:%d)"` at INFO level
- Return filtered list
